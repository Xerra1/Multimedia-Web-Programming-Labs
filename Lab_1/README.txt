1. Make sure you have node.js installed.
2. Using cmd/cli, navigate to the folder.
3. Run 'npm install' to install the dependencies. 
4. Open the backend.js and edit the first parameter path to where it is located on your machine.
4. Go back to cmd/cli (make sure the directory is correct) and run 'node backend.js'
5. Navigate to localhost:8081/lab1.html
6. Hit ctrl + c to stop listening.
7. If you can't get it to run or need further information, please email izwanharris@gmail.com for assistance.
